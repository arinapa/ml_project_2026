# Предсказание академической успеваемости студентов

## Краткое полутехническое описание проекта   
Система на основе методов машинного обучения предсказывающая академическую успеваемость студента на 
следующий модуль/семестр, включая вероятность отчисления.  
Модель использует исторические данные о студентах: успеваемость, посещаемость, академическую нагрузку.    

На основе этих данных система решает задачу линейной регрессии – прогнозирование оценок по предметам, классификации – определение категории студента: успешно проходит обучение, находится в зоне риска, высокая вероятность отчисления.
