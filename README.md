# Предсказание академической успеваемости студентов

## Краткое полутехническое описание проекта   
Система на основе методов машинного обучения предсказывающая академическую успеваемость студента на 
следующий модуль/семестр, включая вероятность отчисления.  
Модель использует исторические данные о студентах: успеваемость, посещаемость, академическую нагрузку.    

На основе этих данных система решает задачу линейной регрессии – прогнозирование оценок по предметам, классификации – определение категории студента: успешно проходит обучение, находится в зоне риска, высокая вероятность отчисления.   

## Описание блокнотов

* data_analysis - анализ исходных данных и выявление проблем
* xgboost - обучение модели классфикации xgboost и метрики
* predict_status - обучение модели классфикации(предсказания отчисления) и сравнение метрик
* predict_avr_grade_base - обучение моделей линейной регрессии (предсказание оценки на следующий модуль) на датасете, построенном жадным методом и сравнение метрик
* predict_avr_grade_fill_na -  обучение моделей линейной регрессии (предсказание оценки на следующий модуль) на датасете с заполнением пропусков и сравнение метрик

* src/data_processing/building_dataset -- функции для построения датасета  
* src/data_processing/data_processing -- функции для очистки датасета
