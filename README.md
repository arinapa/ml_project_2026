# Предсказание академической успеваемости студентов

## Краткое полу-техническое описание проекта   
Система на основе методов машинного обучения, которая будет предсказывать академическую успеваемость студента на 
следующий модуль/семестр, включая вероятность 
отчисления. Модель использует исторические данные о студентах, включая успеваемость, посещаемость, академическую нагрузку.    

На основе этих данных система решает задачу линейной регрессии – прогнозирование оценок по предметам, классификации – определение категории студента: успешно проходит обучение, находится в зоне риска, высокая вероятность отчисления.
